1. Bin folder contains the DLL for AJAX control toolkit, just add it to your toolbox and use the controls generating through it.

2. Add readable and meaningful comments to your code in standard format as follows.
		/// <summary>
        /// get random password
        /// </summary>
        /// <param name="min">min value(lower value)</param>
        /// <param name="max">max value(upper value)</param>
        /// <returns>integer value</returns>
        public int GetRandomNumber(int min, int max)
        {
            Random random = new Random();
            return random.Next(min, max);
        }
